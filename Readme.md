
# Algorithms

This repository contains code snippets and libraries that I have implemented for my personal purposes...

It is very likely that most of the programs made here contradict every rule of software architecture and design pattern. This is because I make them in my spare time, to fulfill very specific and temporary needs, or just to hang out, so these programs do not take effort in their structure, only in making them work.

You will most likely see clutter here, because one of the rules I have for this repository is not to organize the commits.

In case you haven't noticed, this is **NOT** a serious project, so do not use the modules/packages published here, in fact the import path may change without notice.

## History

I usually made some mini-projects to practice in some language and then I deleted them, but, now I consider that it is better to leave that code published somewhere, who knows, maybe I will need them in the future?
